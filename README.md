## Install
```
npm install
```

## Contributing

To contribute, work on the source files by adding new Algorithms to existing service or create a new one.

### Build scripts

##### `npm run build`

Builds the client web page

##### `npm run start`

Runs the server

##### `npm run client-test`

Development

##### `npm run dev`

### Debugging scripts

Development debugging

##### `npm run dev-brk`

Unit Test debugging

##### `npm run test-client`
